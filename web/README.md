Web client
==========

Test using [react](http://facebook.github.io/react/index.html)

Usage
-----

```
npm install express
node server.js
```

Start `apollo-webserver` before:

``` 
../apollo-webserver -config ../etc/apollo.conf &
```

Start `express` server:

```
node server.js
```

Point your browser to http://localhost:3000/.
